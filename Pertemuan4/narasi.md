1. Apa yang Anda pahami tentang Naming Convention?
Naming convention adalah aturan penamaan variabel dalam java di mana sebuah variabel memiliki aturan khusus dalam penamaan variabel dan menggunakan nama yang deskriptif juga bermakna. 

2. Mengapa penamaan variabel dan class penting dalam pemrograman?
Penamaan variabel dan class penting karena untuk memudahkan dalam mengakses dan memanggil variabel/class nantinya dalam kode. Mengikuti aturan penamaan variabel dan class penting untuk diketahui karena itu adalah sebuah standar penulisan kode. Hal ini memudahkan developer lain untuk memahami kode saat bekerja yang terdiri dari beberapa orang. Selain itu, penamaan aturan penamaan penting agar lebih mudah memahami variabel. 

3. Bagaimana operator aritmatika bekerja dalam Java?
+ untuk perrtambahan
- untuk pengurangan
/ untuk pembagian (untuk hasil bagi desimal salah satu operand harus float/double (desimal))
* untuk perkalian
% untuk sisa bagi

++x = tambah dulu
x++ = menggunakan x setelah itu ditambah 1
--x = kurang dulu
x-- = menggunakan x setelah itu dikurang 1

+=, contoh c += 5 sama dengan c = c + 5
-=, contoh c -= 5 sama dengan c = c - 5
*=, contoh c *= 5 sama dengan c = c * 5
/=, contoh c /= 5 sama dengan c = c / 5
%=, contoh c %= 5 sama dengan c = c % 5

untuk menghitung variabel yang memiliki tipe data yang berbeda dapat menggunakan konversi
double value1 = 5.1
int value2 = 5
int hasilKali = (int)(value1 * value2)
hasilKali akan menghasilkan hasil bertipe integer.

terdapat prioritas operator, operasi perhitungan akan dioperasikan sesuai dengan urutan prioritas operator. Namun, lebih baik untuk menggunakan () untuk memperjelas.

4. Contoh penerapan yang anda pahami dari latihan yang dikerjakan
Dalam konversi tipe data bisa saja hasilnya berbeda. Penggunaan () dalam proses konversi tipe data sangat penting dan harus diperhatikan agar hasil operasi sesuai dengan yang dibutuhkan.

5. Kesulitan yang dialami 
Saat memahami konversi tipe data terdapat error karena kurangnya (), karena itu harus memerhatikan error message untuk memahami letak error.
